# 社交内置浏览器防封 · 简易引导页

一个简洁的静态页，满足两种状态：
- 社交软件内置浏览器中：显示提示与分步引导，指导用户“在浏览器打开”。
- 系统浏览器中：自动无痕跳转到目标链接。

## 使用

打开 `index.html` 即可：
- 在社交软件内打开时：显示引导提示。
- 在系统浏览器打开时：自动跳转到目标链接。

目标链接在 `main.js` 的 `TARGET_URL` 中设置为：
`https://kefu.qtxbr.cn/`

## 本地预览

在当前目录运行：
```
python3 -m http.server 8000
```
访问：
`http://localhost:8000/index.html`

## 说明
- 社交软件内置浏览器通常无法强制调起系统浏览器，因此采用“提示引导”的方式；一旦在系统浏览器打开本页，将自动跳转到目标链接。
